Question 1: Explain the difference between an API integration error and a logic error.

API Integration Error:
    This happens when two systems (frontend and backend) cannot communicate properly. They either can't find each other or they speak a different "language".

Logic Error:
    The code compiles and the communication works fine but the computer does the wrong thing because the instructions we wrote are logically incorrect. The code runs but doesn't behave as intended.
